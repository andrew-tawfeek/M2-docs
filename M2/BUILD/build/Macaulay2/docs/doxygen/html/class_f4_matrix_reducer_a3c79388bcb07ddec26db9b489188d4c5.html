SparseMatrix F4MatrixReducer::reduceToBottomRight ( const QuadMatrix & matrix)

Reduces the bottom rows by the top rows and returns the bottom right submatrix of the resulting quad matrix. The lower left submatrix is not returned because it is always zero after row reduction.
