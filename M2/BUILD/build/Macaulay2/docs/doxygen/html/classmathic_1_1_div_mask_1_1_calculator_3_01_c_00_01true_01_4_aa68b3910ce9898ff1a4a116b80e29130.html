void mathic::DivMask::Calculator< C, true >::rebuild ( Iter begin,

Change the meaning of the bits in the div masks produced by this object to something that will likely work well for entries in [begin, end). All div masks will have to be recomputed after this. Mixing div masks computed before a call to rebuild() with ones after has unpredictable results.
