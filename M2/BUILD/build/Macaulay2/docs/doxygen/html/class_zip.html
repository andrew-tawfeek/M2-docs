Detailed Description

template<class Iterator1, class Iterator2>
class Zip< Iterator1, Iterator2 >

Zip ties two iterators together into a single iterator. The point is to enable the zip() function defined further down in this header.

Note that equality is defined ONLY by equality of the first iterator!
