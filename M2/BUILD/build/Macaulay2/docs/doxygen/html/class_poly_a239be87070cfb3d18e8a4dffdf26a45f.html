◆ isMonic()

bool Poly::isMonic ( ) const
inline

Returns true if the polynomial is monic. A polynomial is monic if the coefficient of the leading monomial is 1. If you are asking this question about a polynomial, that likely means that you are expecting the polynomial not to be zero. So it is an error to ask if the zero polynomial is monic - you'll get an assert to help pinpoint the error.
