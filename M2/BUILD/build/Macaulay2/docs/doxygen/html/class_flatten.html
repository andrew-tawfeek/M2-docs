Detailed Description

template<class OuterIterator>
class Flatten< OuterIterator >

Flatten is an iterator that iterates through each range in a range of ranges. The point is to enable the flatten() function defined further down in this header. This iterator is invalidated whenever the outer range or any of the inner ranges invalidate any iterator or change their number of elements.
