Detailed Description

template<template< typename > class Queue>
class ReducerHash< Queue >

The simplest reducer using a hashtable. All terms are in the hash table and in the queue. There are no duplicates as new terms are looked up in the hash table before insertion into the queue. The coefficient is stored in the hash table.
