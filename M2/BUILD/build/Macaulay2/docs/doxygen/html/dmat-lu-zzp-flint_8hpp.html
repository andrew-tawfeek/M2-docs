Detailed Description

DMatLinAlg<M2::ARingZZpFlint> — dense Z/p LU routed through FLINT nmod_mat_*.

Specialises DMatLinAlg for the FLINT-backed Z/p aring. Each operation forwards directly to the matching nmod_mat_* routine: rank becomes nmod_mat_rank, determinant becomes nmod_mat_det, LU becomes nmod_mat_lu, and so on. The underlying nmod_mat_t carries Z/p entries packed as mp_limb_ts with the precomputed ninv reciprocal that FLINT uses for fast Barrett-style modular reduction, so the inner loop never performs an integer division.

The header is #included exactly once from dmat-lu.hpp; the M2/gc-include.h shim must precede the FLINT include so that flint_malloc routes through bdwgc, and the diagnostic pragmas silence FLINT-internal conversion warnings. For very small primes the FFLAS-FFPACK alternative (dmat-lu-zzp-ffpack.hpp) wins via BLAS dispatch; this path is the right choice for medium primes where the BLAS double trick no longer applies.
