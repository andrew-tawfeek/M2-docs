232 Matrix /* or null */ * rawSubduction(int numparts,
233 const Matrix *M,
234 const RingMap *F,
235 Computation *C);
236 /*
237 Perform a subalgebra reduction of the entries of the one row matrix M.
238 C should be a GB computed in high enough degree to handle the elements of M,
239 of an ideal of the form y_i - m_i (m_i is the lead monomial of f_i).
240 F should be a ring map R --> R, sending y_i to f_i.
241 M should be a matrix over the ring R, usually only involving the variables in the f_i.
242 R should be a ring containing the variables of the f_i, and the variables y_i,
243 with a monomial order eliminating the first set of variables (which is 'numparts' parts of the
244 monomial ordering).
245 numparts: number of parts in the monomial order of the original ring.
246 The resulting matrix will have no monomials which are in the subalgebra
247 generated by the monomials m_i, and each entry of M and the corresponding entry of the
248 result differ by an element of the subalgebra generated by the f_i.
249 */
250
251 M2_bool rawIdealOfPoints(const Ring *R,
252 const MutableMatrix *Pts,
253 Matrix /* or null */ ** result_GB,
254 Matrix /* or null */ ** result_std_monoms);
255 /* Returns false if an error occurred.
256 Input: R: a polynomial ring of the form K[x1,...,xn]
257 Pts: an n by d matrix over K.
258 Action: Compute the ideal of the points in n-space
259 given by the columns of 'Pts'
260 Output: result_GB: the GB of this ideal
261 result_std_monoms: the standard monomials (1 by d matrix)
262 Question: should this return the separators as well?
263 */
