Detailed Description

template<template< typename > class Queue>
class ReducerPack< Queue >

Keep a priority queue with entries that represent a term of a polynomial times a multiplier. When an entry is popped, move on to the next term of the polynomial, if any, and push that back into the queue. The idea is that this reduces the number of elements in the queue, leading to faster queue operations. Memory is also saved compared to expanding each new polynomial with n terms into n entries in the queue.
