Detailed Description

template<class C>
class mathic::KDTree< C >

An object that supports queries for divisors of a monomial using a KD Tree (K Dimensional Tree). See DivFinder.h for more documentation.

Extra fields for Configuration:

bool getSortOnInsert() const Return true to keep the monomials in leaves sorted to speed up queries.

size_t getLeafSize() const Return the fixed maximal size of a leaf.

static const bool AllowRemovals If false, it is an error to call methods that remove elements from the data structure. This can be a slight speed up in some cases. Clear and rebuild is still allowed even if this field is false.
