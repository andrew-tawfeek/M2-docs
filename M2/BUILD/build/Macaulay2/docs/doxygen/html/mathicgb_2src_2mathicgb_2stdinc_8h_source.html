188make_unique function. It's pretty easy to do, too:
189
190template<typename T, typename ...Args>
191std::unique_ptr<T> make_unique( Args&& ...args )
192{
193 return std::unique_ptr<T>( new T( std::forward<Args>(args)... ) );
194}
195
196Unfortunately, MSVC does not have variadic templates, so this turns
197into the monstrosity of overloads below. At least they got the perfect
