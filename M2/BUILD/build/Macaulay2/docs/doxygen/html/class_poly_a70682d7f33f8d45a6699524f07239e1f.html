◆ polyWithTermsDescending()

Poly Poly::polyWithTermsDescending ( )
inline

Returns a polynomial whose terms have been permuted to be in descending order.

Making the copy is not wasteful, because doing the permutation in-place would be require swapping monomials which is slow if they are large. The returned object is not copy (return value optimization) and using move assignment this code will only create the single copy of a Poly p that is necessary to avoid an in-place operation:

p = p.polyWithTermsDescending()
