Detailed Description

template<typename T>
class Range< T >

An object that combines two iterators into a range suitable for use with C++11's range for. It is most conveniently used with the function range. For example:

std::vector<int> v; for (int x : range(v.begin(), v.end()) {}

std::vector already has begin and end members, so range() is not necessary here. range() is useful when a class does not have begin or end members, or if it offers several different ranges that can be iterated through - then the default range can only offer one of those ranges.
