template<class T>
FixedSizeMonomialMap< T >::FixedSizeMonomialMap ( const size_t requestedBucketCount,
FixedSizeMonomialMap< T > && map )
inline

Construct a hash table with at least requestedBucketCount buckets and insert the elements from the parameter map.

The parameter map remains a valid object that can satisfy queries. However, it is an error to call non-const methods on the map other than the destructor. Also, insertions into *this may or may not be reflected for queries into map and some of the entries currently in map will disappear.
