105 // memory space for monomials and words for F4 matrix.
106 // PreRow is left, index of poly, right, prevReducer
107 // where the entries depend on the value of prevReducer.
108 // prevReducer = false: if index \geq 0, then left*mGroebner[index]*right
109 // is the lead term of a reducer.
110 // if index < 0, then left*mInput[-index-1]*right
111 // is the lead term of a reducer.
112 // prevReducer = true : left*mRows[i]*right is the lead term of a reducer
113 // the int at the end is the index of the PreRow in the
114 // corresponding vector it belongs to, which will eventually
115 // be the corresponding row.
