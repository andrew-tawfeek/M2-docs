Detailed Description

template<class Stream>
class mgb::IdealStreamChecker< Stream >

Use this class to check that you are following the correct protocol for calling methods on an ideal stream. This has significant overhead so it is not recommended for production use. If you have built the MathicGB library in debug mode then this is already automatically used for GroebnerInputIdealStream.
