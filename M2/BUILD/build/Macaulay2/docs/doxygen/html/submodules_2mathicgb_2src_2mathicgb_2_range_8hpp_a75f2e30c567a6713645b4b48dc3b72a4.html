Constructs a half-open interval of integers [begin, end) that is compatible with range-for.

Example: for (auto x : intRange(5, 10) std::cout << ' ' << x; The output will be " 5 6 7 8 9". See indexRange() for further examples of how this sort of thing can be useful.
