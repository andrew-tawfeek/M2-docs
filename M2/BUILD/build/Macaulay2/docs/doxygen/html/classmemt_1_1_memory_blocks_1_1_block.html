Detailed Description

A block owns a range of memory [begin(), end()) and keeps track of a position within the range [begin(), end()] given by position(). It is possible for a block to own no memory, in which case begin(), end() and position are null - we call this a null block object. Do not create your own blocks - their lifetime is handled by MemoryBlocks.
