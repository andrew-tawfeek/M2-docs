Detailed Description

template<class E, bool HC, bool SH, bool SO>
class MonoMonoid< E, HC, SH, SO >

Implements the monoid of (monic) monomials with integer non-negative exponents. Exponent must be an unsigned integer type that is used to store each exponent of a monomial.
