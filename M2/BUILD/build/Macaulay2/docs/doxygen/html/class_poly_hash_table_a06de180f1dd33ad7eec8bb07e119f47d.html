void PolyHashTable::clear ( )
inline

Removes all elements and optimizes internal resources. This is fast if there are no elements, so if you know that there are no elements and that many operations have happened since the last clear, then call clear for better cache performance. If there is even one element, then this takes linear time in the number of buckets.
