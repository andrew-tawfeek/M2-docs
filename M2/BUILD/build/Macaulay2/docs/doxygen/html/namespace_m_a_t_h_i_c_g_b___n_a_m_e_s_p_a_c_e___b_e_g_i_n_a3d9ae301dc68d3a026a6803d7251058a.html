bool MATHICGB_NAMESPACE_BEGIN::fileExists ( const std::string fileName)

Returns true if the file exists - or more precisely if it can be opened for reading. Using this function to create a file only if it does not exist implies a race condition in that the file could have been crated after checking if it exists and before (re)creating it. Do not use this approach if that is not acceptable. The advantage here is that this is portable. There could be a solution with freopen, but unfortunately freopen is allowed to fail on any change to the mode so it is not a portable solution.
