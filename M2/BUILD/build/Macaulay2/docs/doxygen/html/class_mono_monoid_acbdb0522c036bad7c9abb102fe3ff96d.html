◆ externalVar()

template<class E, bool HC, bool SH, bool SO>
VarIndex MonoMonoid< E, HC, SH, SO >::externalVar ( const VarIndex var) const
inline

The indices of variables might be permuted in this monoid in order to implement certain monomial orders in a fast way. This method returns the unpermuted index that gets permuted to var. Knowing this mapping is necessary when importing monomials from and when exporting monomials to a party that does not use this scheme. For example when writing a monomial to a file in a format that is intended to be readable by other programs or by a human.
