32// Class: Subsets
33// Manipulate p-subsets of 0..n-1, for fixed n, but possibly several p.
34//
35// Encoding and decoding is done as follows:
36// Consider e.g. 3-subsets:
37// These are
38// 0 = 0 1 2
39// 1 = 0 1 3
40// 2 = 0 2 3
41// 3 = 1 2 3
42// 4 = 0 1 4
43// 5 = 0 2 4
44// 6 = 0 3 4
45// 7 = 1 2 4
46// 8 = 1 3 4
47// 9 = 2 3 4
48// ... (if there are more than 5 elements in the set).
49// Notice that the encoded value does not depend on the number of elements
50// [a0, a1, ..., a(p-1)] becomes
51// the integer binomial(a0,1) + binomial(a1,2) + binomial(a2,3) + ...
52// example: 10 = 0 1 5. This is binomial(0,1) + binomial(1,2) + binomial(5,3) =
53// 10
54// example: 9 = 2 3 4. 9 == 2 + binomial(3,2) + binomial(4,3) = 2 + 3 + 4
55
