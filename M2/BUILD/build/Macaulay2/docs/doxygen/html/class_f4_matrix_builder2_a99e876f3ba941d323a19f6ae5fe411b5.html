◆ addSPolynomialToMatrix()

void F4MatrixBuilder2::addSPolynomialToMatrix ( const Poly & polyA,
const Poly & polyB )

Schedules a row representing the S-polynomial between polyA and polyB to be added to the matrix. No ownership is taken, but polyA and polyB must remain valid until the matrix is constructed.

Currently, the two monomials must be monic, though this is just because they happen always to be monic so there was no reason to support the non-monic case.
