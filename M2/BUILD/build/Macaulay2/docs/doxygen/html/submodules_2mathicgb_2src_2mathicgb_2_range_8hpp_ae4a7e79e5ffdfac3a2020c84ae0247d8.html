template<class Iterator>
auto indexRange ( const Iterator & begin,
const Iterator & end )->decltype(zip(range(begin, end), intRange()))

Adds indices to the elements of the range, specifying the offset of each element in the range. Equivalent to zip(range(begin, end), intRange()).

Example: const char*[] strs = {"hello", "world", "!"}; for (const auto& p : indexRange(strs)) std::cout << ' ' << p.first << p.second;

The output will be " hello0 world1 !2".
