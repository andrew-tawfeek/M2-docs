Detailed Description

template<class W>
class MonoOrder< W >

Class used to describe an monomial order and/or a module monomial order. Use this class to construct a monoid. The monoid does the actual comparisons. Monomials must be preprocessed by MonoProcessor - otherwise the ordering may not be correct. MonoProcessor also offers additional parameters for making orders.
