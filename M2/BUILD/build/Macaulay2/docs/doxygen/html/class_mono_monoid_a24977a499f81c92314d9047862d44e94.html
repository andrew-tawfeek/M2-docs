template<class E, bool HC, bool SH, bool SO>
void MonoMonoid< E, HC, SH, SO >::setExternalExponents ( const Exponent * exponents,
MonoRef mono ) const
inline

Sets all the exponents of mono from an external/unpermuted array. exponents must point to an array of size varCount(). After this, exponent(mono, var) is exponents[externalVar(var)]. The value of exponents[var] becomes the exponent of internalVar(var). Does not set the component.
