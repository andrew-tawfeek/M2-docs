◆ fromArena()

bool memt::Arena::fromArena ( void const * ptr)

Returns true if ptr is within the range of any memory buffer that has been allocated from this arena and that has not yet been deallocated. Pointers that are one-past-the-end of an allocated buffer are not within the range, but they may be inside a subsequent buffer and so may still yield a return value of true. Also, the allocated range may be larger than requested due to alignment in which case the pointer that is one-past-the-end would be inside the range. This method is useful for debugging and testing.
