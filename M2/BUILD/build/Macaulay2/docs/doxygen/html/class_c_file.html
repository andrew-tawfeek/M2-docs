Detailed Description

RAII handle for a C FILE*.

The purpose of using the C IO interface instead of iostreams is that the former is faster to a ridiculous degree. This class wraps the C IO interface to be more useful in a C++ context. For example the file is automatically closed in the destructor and if the file cannot be opened then an exception is thrown instead of returning a null pointer.
