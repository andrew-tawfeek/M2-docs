Detailed Description

template<class AbstractProduct>
class mathic::NameFactory< AbstractProduct >

A NameFactory takes a name and then creates an instance of a class that has been previously registered under that name. This is done in a general way using templates.

None of this is very efficient currently. However, the interface can be implemented much more efficiently if that becomes necessary.
