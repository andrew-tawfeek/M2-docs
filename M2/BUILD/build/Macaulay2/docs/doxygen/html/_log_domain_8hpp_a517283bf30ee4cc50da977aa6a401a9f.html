Runs the code in the following scope delimited by braces {} if the indicated logger is enabled for streaming - otherwise does nothing. Within the following scope there is a local reference variable log that refers to the indicated logger and a local reference variable stream that refers to log.stream().

Example: MATHICGB_IF_STREAM_LOG(MyDomain) { std::string msg; expensiveFunction(msg); stream << msg; // or log.stream() << msg; }
