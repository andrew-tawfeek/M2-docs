enum mgb::GroebnerConfiguration::BaseOrder
Enumerator
RevLexDescendingBaseOrder 

Reverse lexicographic order with x_1 > x_2 > ... > x_n. Can be described as rev-lex-descending or rev-lex-from-right. This is what people usually mean when they say rev-lex.

LexDescendingBaseOrder 

Lexicographic order with x_1 > x_2 > ... > x_n. Can be described as lex-descending or lex-from-left. This is what people usually mean when they say lex.

RevLexAscendingBaseOrder 

Reverse lexicographic order with x_1 < x_2 < ... < x_n. Can be described as rev-lex-ascending or rev-lex-from-left.

LexAscendingBaseOrder 

Lexicographic order with x_1 < x_2 < ... < x_n. Can be described as lex-ascending or lex-from-right.
