Detailed Description

template<class T, bool FreeMemory>
class memt::ArenaVector< T, FreeMemory >

Works as std::vector, except it cannot grow its capacity and the memory is taken from an Arena.

Only frees its memory on destruction if FreeMemory is true. In that case that memory must be the top allocation on the Arena.

Always calls the destructors of contained entries even when FreeMemory is false.
