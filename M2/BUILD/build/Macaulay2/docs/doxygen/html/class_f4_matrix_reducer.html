Detailed Description

Class that reduces an F4 matrix represented as a QuadMatrix. The answer that you get is the submatrix that contains new pivots.

All QuadMatrix parameters passed into methods on this class are assumed to have a permutation of the top rows and left columns so that the top left matrix is upper unitriangular. In this way the lower left part of the matrix becomes all-zero after row reduction.
