157// to create the frame:
158
159// 1. Fill in level 0 - (Take the degree information from the target mInputModGB)
160// 2. Fill in level 1 - (This is really just the info on the entries of mInputModGB)
161// This is more difficult the GB is not known out to that point yet.
162// 3. Fill in level 2 - This requires us to find all the spairs on module elements vs
163// elements of the ring Groebner basis
164// 4. Rinse and repeat.
165
166// detailed notes on a basic example
167// consider the right ideal (yyx,yxx)R with R = QQ<|x,y|>/(x*y - y*x)
168
169// Step 1:
170// level 0: right module generated by e_1
171
172// Step 2:
173// level 1: f_1 --> e_1yyx, f_2 --> e_1yxx
174
175// other gb elements? (e_1yyx) * y - (e_1 yy)*(x*y - y*x) = e_1 yyyx
176// (e_1yxx) * y - (e_1 yx)*(x*y - y*x) = e_1 yxyx ~> e_1yyxx - (e_1yyx)*x = 0
177
178// f_3 --> e_1 yyyx, etc (another f_j for each e_i y^j x for j >= 3)
179
180// Level 2:
181
182// g_1 --> f_1.y = e_1.yyx.y <~~> e_1.yyxy = e_1.yyyx, now have to add f_3 to level 1
183
184// g_1 --> f_1.y = e_1.yyx.y <~~> f_1.y - f_3 Claim: this element maps to zero under the previous map
185// g_1 will have label: e_1.yyx.y e_1yyxy - e_1yyyx = e_1yy.(xy - yx) = 0
186
187// g_2 --> f_2.y = e_1.yxx.y <~~> f_2.y - f_1.x e_1yxxy - e1yyxx = 0
188// g_2 will have label: e_1.yxx.y
189
190// g_3 --> f_3.y = e_1.yyyx.y <~~> f_3.y - f_4 (etc)
191// g_3 will have label: e_1.yyyx.y
192
193// for a term, we have: label - product of all leadTerms all the way back to level 0, e.g. e_1.yyyx.y (without the dots)
194
195// to compare
196// g_1.m_1; g_1 = f_1.v_1
197// g_2.m_2; g_2 = f_2.v_2, f_1 and f_2 are further back etc
198
199// first compare total monomials of g_1.m_1 and g_2.m_2. if there is a winner, take that term.
200// if there is a tie at this stage,
201
202// example:
203// level 0: e_1
204// level 1: f_1 = e_1.xy, f_2 = e_1.y^3, f_3 = e_1.yx
205// level 2: g_1 = f_1.xyy g_2 = f_1.xxy, g_3 = f_2.x
206
207// How to compare f_i.m and f_j.n: f_1.xxx vs f_2.xy
208// labels are e_1.xyxxx e_1.yyyxy (compare here). One is a winner.
